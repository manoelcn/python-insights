# python-insights
Esse projeto faz parte de um estudo de análise de dados utilizando `python`, focado em entender os motivos pelos quais os clientes de uma empresa de grande porte estão cancelando seus serviços. O objetivo principal é identificar padrões de cancelamento e propor ações estratégicas para melhorar a retenção de clientes.

## Contexto do desafio
Uma empresa percebeu que a maioria da sua base total de clientes está inativa. Para reverter esse cenário, foi realizada uma análise exploratória para identificar as causas principais do cancelamento e calcular o impacto de possíveis medidas preventicas no faturamento e na taxa de retenção.

## Arquivos do repositório
- `main.ipynb`: notebook com todo o código da análise
- `cancelamentos.csv`: base de dados utilizada
- `requirements.txt`: lista de bibliotecas necessárias para rodar o projeto.
